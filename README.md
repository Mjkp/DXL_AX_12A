# DXL_AX_12A
control pipeline from Unity to pi, controlling dynamixel servos via udp protocol


## Prerequisite
- dynamixel sdk
- python-osc (python 3)


## Material
- Raspberry pi 
- Robotis Dynamixel AX_12A
- U2D2
- SMPS2Dynamixel
_ 12V DC Adaptor
- Micro USB cable
- Micro USB (female) cable (if using pi zero)
